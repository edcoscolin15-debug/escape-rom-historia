Jogos Pedagógicos em HTML - Aulas de História

Bem-vindo ao repositório de jogos educativos! Este projeto reúne diversas atividades interativas (Escape Room, Quiz, Tabuleiro, Campo minado, Agentes do Tempo) desenvolvidas para tornar o ensino de História mais dinâmico e envolvente, voltado especialmente para alunos do 6º ao 9º ano.

📖 Guia prático de uso: Escape Room

Este jogo foi desenvolvido para transformar o aprendizado em uma experiência imersiva e dinâmica. Através de uma interface baseada em "Escape Room", os alunos assumem o papel de protagonistas de eventos históricos, precisando decifrar enigmas e responder questões para avançar nas etapas do jogo.

⚙️ Regras do Jogo

  1. O Objetivo: A equipe deve completar todas as "Salas" (perguntas) do tema escolhido para obter a liberdade/vitória.
  2. O Cronômetro (Tensão): Cada sala possui um tempo limite. Se o tempo zerar, o sistema aplica uma penalidade severa: o progresso é resetado e a equipe retorna automaticamente para a "Sala 1". Isso incentiva a agilidade e o trabalho em equipe.
  3. Validação Inteligente: O sistema não é rígido. Ele utiliza a Distância de Levenshtein para perdoar pequenos erros de digitação.
    • Se o aluno errar uma letra, o sistema aceita a resposta, mas exibe um aviso educativo mostrando a grafia correta para correção imediata.
  4. Trabalho Colaborativo: O jogo é ideal para ser jogado em grupos, onde um aluno lê a narrativa enquanto outro manipula o dispositivo.

🛠 Como Utilizar

  • Painel de Controle: Na tela inicial, o botão "Editor e Criador do Jogo" dá acesso total ao motor do jogo.
  • Importação/Exportação: Você pode criar seus temas, exportá-los em formato JSON para o seu computador (como backup) ou importar arquivos de outros professores para expandir sua biblioteca de desafios.
  • Flexibilidade: Ao criar uma pergunta, você define:
    • O cenário (história imersiva).
    • A pergunta e a resposta correta.
    • O tempo limite (ajuste conforme a complexidade).
    • A Tolerância de Erros (quantas letras erradas o aluno pode digitar e ainda assim avançar).

🤖 Prompt para Criar Novos Desafios com a IA

Para gerar novos roteiros prontos para importação, utilize o seguinte prompt na sua ferramenta de IA. Copie e cole este modelo:

"Atue como um especialista em design de jogos pedagógicos para o Ensino Fundamental II. Crie um arquivo JSON estruturado para um 'Escape Room' temático sobre [INSIRA O TEMA AQUI, EX: A Revolução Industrial]. Requisitos: 1. Crie 10 a 15 salas (perguntas) em sequência lógica. 2. Antes de cada pergunta, escreva um parágrafo narrativo (história imersiva) que situe o aluno no contexto histórico. A narrativa deve ser contínua ao longo das salas. 3. Cada pergunta deve ter um claro valor pedagógico e nível de dificuldade entre DEFINA AQUI O NÍVEL E O PÚBLICO ALVO. 4. Adicione um campo 'timeLimit' (entre 45 e 120 segundos) e um campo 'errorTolerance' (0 a 3 letras). 5. Formate a saída como um JSON válido, pronto para importação direta, contendo os campos: 'roomName', 'description', 'question', 'answer', 'timeLimit', 'errorTolerance'."

💡 Dicas Pedagógicas para o Professor

  • Mediação: Como o jogo salva o progresso via LocalStorage no próprio navegador, você pode permitir que os alunos comecem uma missão e a terminem em uma aula subsequente, caso o tempo da aula atual acabe.
  • Personalização: Use o campo "História/Cenário" para inserir curiosidades históricas que não estão nos livros didáticos, tornando a narrativa mais instigante.
